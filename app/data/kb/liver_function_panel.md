# 肝功能检验项目解读要点

> 本文档为 MediRead RAG 知识库种子样本（节选自《临床检验项目大全》《中国成人体检基本项目专家共识》要点改写），用于本地/演示环境跑通检索 + 引用链路。生产环境请用授权完整版替换。

## ALT（丙氨酸氨基转移酶）

- **意义**：主要存在于肝细胞胞浆，肝细胞损伤时释放入血。
- **参考范围**：成人 7–40 U/L（男性可略高）。
- **轻度升高（< 2 ×上限）**：常见于脂肪肝、酒精性肝病、药物性损伤、剧烈运动后、轻度病毒性肝炎。建议复查与生活方式调整。
- **中度升高（2–5 ×上限）**：需排查急性肝炎、胆道梗阻、自身免疫性肝病；建议消化内科 / 肝病门诊就诊。
- **明显升高（> 5 ×上限）**：警惕急性病毒性肝炎、急性中毒性肝损伤；建议尽快专科就诊。

## AST（天门冬氨酸氨基转移酶）

- **意义**：除肝脏外亦广泛存在于心肌、骨骼肌、肾、红细胞。
- **参考范围**：13–35 U/L。
- **AST/ALT 比值**：> 1 提示酒精性肝病或肝硬化；< 1 多见于急性病毒性肝炎。
- 升高且 ALT 正常时，需结合 CK / LDH 排查心肌损伤。

## GGT（γ-谷氨酰转移酶）

- **意义**：胆道系统疾病敏感指标，亦受酒精影响明显。
- **参考范围**：成人男性 11–50 U/L，女性 7–32 U/L。
- 联合 ALP 升高提示胆汁淤积；单独 GGT 升高常见于酒精摄入过量。

## ALT + AST + GGT 三项联合分析

- 三项同时偏高 → **肝细胞 + 胆道系统受损迹象**，建议肝胆超声 + 消化内科就诊。
- ALT > AST + GGT 正常 → 单纯肝细胞损伤倾向（脂肪肝 / 病毒 / 药物）。
- GGT 突出升高 → 优先排查饮酒、胆道、药物。

## 生活方式建议（通用）

- 戒酒或严格限酒
- 高脂高糖饮食控制，体重管理
- 慎用对乙酰氨基酚等肝毒性药物
- 2–4 周复查动态观察
