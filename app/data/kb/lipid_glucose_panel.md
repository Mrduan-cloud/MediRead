# 血脂血糖检验项目解读要点

> 本文档为 MediRead RAG 知识库种子样本，用于演示环境。生产环境请用授权完整版替换。

## TC（总胆固醇）

- **参考范围**：< 5.2 mmol/L（理想）；5.2–6.2 边缘升高；≥ 6.2 mmol/L 高
- 显著升高需排查家族性高胆固醇血症，建议心内科评估

## TG（甘油三酯）

- **参考范围**：< 1.7 mmol/L 理想；1.7–2.3 边缘升高；≥ 2.3 高；≥ 5.7 极高 → 急性胰腺炎风险
- TG ≥ 5.7 mmol/L 属危急值，建议尽快内分泌 / 消化科就诊

## LDL-C（低密度脂蛋白胆固醇）

- 一般人群目标 < 3.4 mmol/L；高危人群 < 1.8 mmol/L
- 是动脉粥样硬化的主要致病因素之一

## HDL-C（高密度脂蛋白胆固醇）

- 男性 > 1.0 mmol/L、女性 > 1.3 mmol/L 为正常
- 偏低提示心血管风险升高；可通过规律运动 + 健康饮食改善

## 血脂三项联合分析

- TC + LDL-C 同时升高，TG 正常 → 高胆固醇血症
- TG 单独升高 → 多见于代谢综合征 / 胰岛素抵抗
- TG 极高 + 既往饮酒史 → 排查酒精性高甘油三酯血症

## FPG（空腹血糖）

- 正常 3.9–6.1 mmol/L；6.1–7.0 空腹血糖受损；≥ 7.0 mmol/L 提示糖尿病可能
- 仅一次升高需复测，建议加做 OGTT 和 HbA1c 确诊

## HbA1c（糖化血红蛋白）

- 正常 < 6.0%；6.0–6.4% 糖尿病前期；≥ 6.5% 提示糖尿病
- 反映近 2–3 个月平均血糖水平

## 生活方式建议（通用）

- 控制饱和脂肪 < 7% 总能量；增加可溶性纤维（燕麦、豆类、苹果）
- 每周 ≥ 150 分钟中等强度运动
- 戒烟限酒；保证 7+ 小时睡眠
- 3 个月复测血脂血糖；HbA1c 半年一次
