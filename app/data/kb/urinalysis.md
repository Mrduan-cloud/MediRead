# 尿常规检验项目解读要点

> 本文档为 MediRead RAG 知识库种子样本（节选自《全国临床检验操作规程》《中国成人体检基本项目专家共识》要点改写），用于本地/演示环境跑通检索 + 引用链路。生产环境请用授权完整版替换。

## 尿蛋白（PRO，Protein）

- **意义**：尿中蛋白含量，反映肾小球滤过屏障与肾小管重吸收功能。
- **参考范围**：定性 阴性（-）；定量 < 150 mg/24h。
- **弱阳性（±~+）**：可见于剧烈运动、发热、寒冷、直立性蛋白尿等生理性因素，建议清晨复查、避免运动后留尿。
- **持续阳性（≥ +）**：提示肾小球/肾小管病变（如肾炎、糖尿病肾病、高血压肾损害），建议肾内科就诊 + 24h 尿蛋白定量 + 尿微量白蛋白/肌酐比（ACR）。

## 尿糖（GLU，Glucose）

- **意义**：血糖超过肾糖阈（约 8.9–10 mmol/L）时由尿排出。
- **参考范围**：阴性（-）。
- **阳性**：优先排查糖尿病/糖耐量异常（需结合空腹血糖、糖化血红蛋白）；少数为肾性糖尿（血糖正常而肾糖阈下降）。

## 尿潜血 / 隐血（BLD，Blood）

- **意义**：检测尿中红细胞或游离血红蛋白/肌红蛋白。
- **参考范围**：阴性（-）。
- **阳性**：需结合镜检红细胞区分——镜检 RBC 增多提示泌尿系结石/感染/肾炎/肿瘤；镜检阴性而潜血阳性见于血红蛋白尿/肌红蛋白尿（溶血、剧烈运动、横纹肌溶解）。女性应避开月经期留尿。

## 尿白细胞（LEU）与镜检

- **参考范围**：定性 阴性；镜检 白细胞 < 5/HP，红细胞 < 3/HP。
- **白细胞增多**：提示泌尿系感染（尿路感染、肾盂肾炎），常伴尿亚硝酸盐阳性；建议尿培养 + 药敏。

## 尿比重（SG）与酸碱度（pH）

- **尿比重参考范围**：1.003–1.030（晨尿多 > 1.020）。持续偏低提示肾浓缩功能下降或尿崩；偏高见于脱水、高热、糖尿。
- **尿 pH 参考范围**：4.5–8.0（常 5.5–6.5）。受饮食影响大；持续异常需结合肾小管酸中毒、结石成分评估。

## 尿酮体（KET）与胆红素 / 尿胆原（BIL / URO）

- **尿酮体**：阴性。阳性见于饥饿、低碳水饮食、剧烈呕吐、糖尿病酮症（需警惕糖尿病酮症酸中毒）。
- **尿胆红素 / 尿胆原**：胆红素阴性、尿胆原阴性或弱阳。胆红素阳性提示梗阻性/肝细胞性黄疸；尿胆原明显升高见于溶血或肝病，需结合肝功能。

## 尿蛋白 + 尿潜血 + 镜检 联合分析

- 尿蛋白 + 潜血同时阳性、镜检见红细胞与管型 → **肾小球肾炎**倾向，建议肾内科尽快就诊。
- 白细胞 + 亚硝酸盐阳性、伴尿频尿急 → **泌尿系感染**，多可门诊处理。
- 仅潜血阳性而镜检红细胞阴性 → 排查血红蛋白尿 / 肌红蛋白尿，结合溶血指标与 CK。

## 生活方式与复查建议（通用）

- 留取清晨中段尿、避免运动 / 月经期 / 大量饮水后即时留尿，减少假阳性。
- 单项轻度异常优先 1–2 周复查，动态观察比单次结果更有意义。
- 反复或持续异常、或合并水肿/高血压/尿量改变，建议肾内科或泌尿外科专科评估。
