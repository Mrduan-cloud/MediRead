# 肾功能检验项目解读要点

> 本文档为 MediRead RAG 知识库种子样本（节选自《临床检验项目大全》《中国成人体检基本项目专家共识》《慢性肾脏病筛查诊断及防治指南》要点改写），用于本地/演示环境跑通检索 + 引用链路。生产环境请用授权完整版替换。

## 血尿素 / 尿素氮（Urea / BUN）

- **意义**：蛋白质代谢终产物，经肾小球滤过排出，反映肾脏排泄功能，但受饮食、脱水、消化道出血影响较大。
- **参考范围**：成人 2.9–8.2 mmol/L。
- **升高**：肾性（肾功能不全）需结合肌酐判断；肾前性见于脱水、高蛋白饮食、消化道出血、心衰；建议复查并结合肌酐、eGFR。
- **降低**：见于严重肝病、低蛋白饮食、妊娠，临床意义相对小。

## 血肌酐（Cr / SCr，Creatinine）

- **意义**：肌肉肌酸代谢产物，生成稳定、几乎全由肾小球滤过，是评估肾小球滤过功能的核心指标。
- **参考范围**：成人男性 59–104 µmol/L，女性 45–84 µmol/L（受肌肉量影响）。
- **轻度升高**：提示肾小球滤过功能下降，需计算 eGFR 分期；建议肾内科就诊、复查、避免肾毒性药物。
- **明显升高**：警惕急性肾损伤或慢性肾脏病进展，建议尽快专科评估。
- 注意：肌酐对早期肾损伤不敏感（肌酐升高时肾功能常已下降约 50%），早筛建议结合 eGFR 与尿微量白蛋白。

## 血尿酸（UA，Uric Acid）

- **意义**：嘌呤代谢终产物，经肾排泄，与痛风、代谢综合征、肾结石相关。
- **参考范围**：成人男性 208–428 µmol/L，女性 155–357 µmol/L。
- **升高（高尿酸血症）**：与高嘌呤饮食、饮酒、肥胖、肾排泄减少相关；可无症状，亦可诱发痛风性关节炎、尿酸性肾结石。建议低嘌呤饮食 + 多饮水，反复发作或显著升高者风湿免疫/内分泌科就诊。

## 估算肾小球滤过率（eGFR）

- **意义**：基于肌酐（及年龄、性别）的公式估算，是慢性肾脏病（CKD）分期的主要依据。
- **分期参考**：≥ 90 正常（G1）；60–89 轻度下降（G2）；45–59 / 30–44 中度下降（G3a/G3b）；15–29 重度下降（G4）；< 15 肾衰竭（G5）。
- eGFR 持续 < 60 超过 3 个月，或伴尿蛋白/结构异常，即符合 CKD 诊断，建议肾内科规律随访。

## 胱抑素 C（Cystatin C）

- **意义**：不受肌肉量、性别影响，对早期、轻度肾功能下降比肌酐更敏感。
- **参考范围**：约 0.59–1.03 mg/L。
- 升高早于肌酐时，提示早期肾小球滤过功能受损，建议结合 eGFR 综合判断。

## 尿素 + 肌酐 + 尿酸 + eGFR 联合分析

- 肌酐 ↑ 且 eGFR ↓ → **肾小球滤过功能受损**，按 eGFR 分期管理，肾内科就诊。
- 尿素 ↑ 而肌酐 / eGFR 正常 → 多为肾前性（脱水 / 高蛋白 / 消化道出血），先纠正诱因再复查。
- 尿酸单独 ↑ 而肌酐正常 → 高尿酸血症，重点在饮食与生活方式，警惕痛风与尿酸性肾损害。

## 生活方式与复查建议（通用）

- 控制血压、血糖、血尿酸（三者是慢性肾损害的主要可控因素）。
- 避免长期 / 大剂量肾毒性药物（部分 NSAIDs、某些抗生素、含马兜铃酸中药）。
- 低盐、适量优质蛋白饮食，充足饮水；高尿酸者限酒与高嘌呤食物。
- 单项轻度异常 1–3 个月复查；持续异常或合并水肿/尿量改变，肾内科专科评估。
